Avaliação HTML e CSS

  1. O que é a semântica em HTML e qual a sua importância no desenvolvimento de páginas web?

    2. Semântica em HTML significa usar tags que definem o significado
    do conteúdo, como <header>, <footer>, e <article>. Isso melhora
    acessibilidade, SEO, manutenibilidade, e estrutura do código.

  2. Qual é a diferença entre uma classe e um ID em CSS e quando usar cada um?

    3. Classe: Para múltiplos elementos
    ID: Para um único elemento

  3. O que é o conceito de 'box model' no CSS e como ele afeta o layout dos elementos em uma página?

    4. O box model no CSS define a estrutura dos elementos HTML, incluindo o
    conteúdo, o padding (espaço interno), a borda e a margem (espaço externo).
    Ele afeta o layout determinando como o tamanho e o espaçamento dos
    elementos são calculados e como eles interagem com outros elementos
    na página.

  4. Como o Flexbox facilita a criação de layouts responsivos em comparação com métodos antigos, como float?

    5. O Flexbox facilita a criação de layouts responsivos ao oferecer alinhamento e
    distribuição de espaço simples, adaptação automática ao tamanho do contêiner, e
    controle flexível sobre a direção e tamanho dos itens. Isso simplifica o processo
    em comparação com métodos antigos como float, que exigiam ajustes complexos e
    hacks para alcançar o mesmo nível de flexibilidade e adaptação.

  5. O que é um reset CSS e por que é importante incluí-lo no início de uma página?

    6. Um reset CSS remove os estilos padrão dos navegadores, garantindo uma aparência consistente
    dos elementos em diferentes browsers. É importante para garantir que todos os estilos
    comecem de uma base uniforme e evitar inconsistências no layout.
